# Roadmap

## Milestone 1: Deterministic Core

- Static analysis rules.
- Concept prediction rules.
- Runtime traceback explanations.
- Privacy-safe output redaction.
- Execution policy for long-running snippets and UI auto-close timers.
- CLI.
- Local web demo.
- Unit tests and CI.

## Milestone 2: Live Developer UI

- PyQt6 split-view editor.
- Debounced analysis while typing.
- Diagnostics grouped by severity.
- Preview panel for text output and simple charts.

## Milestone 3: Data and UI Preview

- pandas dataframe preview.
- matplotlib/seaborn image preview.
- Tkinter/PyQt6 project detection.
- Style file detection and UI refresh experiments.

## Milestone 4: Optional Explanation Providers

- Optional external explanation provider.
- Deterministic fallback diagnostics.
- Explanation templates and evaluation examples.
- Human-readable review notes for suggested fixes.

## Milestone 5: Portfolio-Grade Demonstration

- Screenshots.
- Demo GIF.
- Example projects.
- Review-friendly README section.
